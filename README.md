# Link-Analysis-using-Vote-Wiki-Dataset
This project focuses on performing link analysis on the Wiki-Vote dataset to identify influential 
nodes (users) in a network. The dataset is modeled as a directed graph where nodes represent 
users and edges represent voting relationships. Two fundamental link analysis 
algorithms-PageRank and HITS (Hyperlink-Induced Topic Search) are implemented to evaluate 
node importance. Additionally, structural properties such as indegree and outdegree are 
computed for each node. The system allows users to input a node ID and retrieve its 
corresponding metrics, including PageRank score, HITS authority and hub values, and degree 
information. This project demonstrates how graph-based ranking algorithms can be applied to 
social network data to uncover influence patterns and user behavior.
